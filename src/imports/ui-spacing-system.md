# Fashion E-Commerce UI Spacing System  
*Layout & Spacing Guidelines*

Create a structured spacing system that ensures visual rhythm, alignment consistency, and a premium editorial feel across the entire store.

Use an **8px base spacing system** for consistency and scalability.

---

# OPTION 1 — Edge-to-Edge Product Grid System

## 1. Global Spacing Unit

Base unit: **8px**

Spacing scale:
- 4px
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px
- 96px

All spacing values must follow this scale.

---

## 2. Global Edge-to-Edge Rule (Content Behavior)

This layout follows a strong **edge-to-edge principle**.

### Core Rule

Main content areas extend fully to the viewport width without horizontal container padding.

- No outer horizontal margins
- No fixed-width centered container
- Content aligns directly to the left and right screen edges
- Full-width visual rhythm

Edge-to-edge applies to:
- Product grids
- Promotional banners
- Image sections
- Sale grids
- Catalog listings

Text-based sections (editorial content, policies, long-form text) may use controlled inner padding for readability.

---

## 3. Header Spacing

### Desktop
- Horizontal padding: 32px–48px
- Vertical padding: 16px–24px
- Distance from hero/banner below: 32px

Header may visually align with edge-to-edge system while maintaining internal padding.

### Mobile
- Horizontal padding: 16px
- Vertical padding: 12px–16px

Header must feel compact but breathable.

---

## 4. Footer Spacing

### Desktop
- Horizontal padding: 48px
- Vertical padding: 48px–64px
- Column gap inside footer: 32px

### Mobile
- Horizontal padding: 16px
- Vertical padding: 32px
- Section spacing: 24px

Footer may remain structured with internal padding but visually align to full width.

---

## 5. Product Card Grid — Edge to Edge

### Core Rule

Product grid is strictly **edge-to-edge horizontally**.

- No horizontal outer margins
- No container padding
- Cards align directly to viewport edges

### Columns

- Mobile: 2 columns
- Small desktop: 4 columns
- Large desktop: 5 columns
- Extra-wide screens: 6 columns

### Horizontal Gaps
- No outer padding
- Internal grid gap: 0px or 1px (if border system requires separation)

### Vertical Gaps
- Row spacing: 24px–32px
- Section spacing above grid: 32px–48px

Vertical rhythm must remain consistent.

---

## 6. Section Spacing

Between major sections:
- 64px–96px (desktop)
- 48px (mobile)

Between title and content:
- 24px–32px

---

## 7. Content Blocks

Text blocks:
- Paragraph spacing: 16px
- Heading to paragraph: 16px–24px
- Heading to section: 32px–48px

Editorial content may use internal padding (32px–48px desktop / 16px mobile) for readability, but outer layout remains edge-to-edge.

---

# Design Direction (Option 1)

This system creates:
- Strong visual impact
- Immersive product presentation
- Modern fashion editorial layout
- Bold, confident structure

The experience must feel immersive, sharp, and product-focused.
