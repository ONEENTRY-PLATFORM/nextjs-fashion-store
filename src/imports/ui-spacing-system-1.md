# Fashion E-Commerce UI Spacing System  
*Layout & Spacing Guidelines*

Create a structured spacing system that ensures visual rhythm, alignment consistency, and a premium editorial feel across the entire store.

Use an **8px base spacing system** for consistency and scalability.

---

# OPTION 1 — Edge-to-Edge Product Grid System

## 1. Global Spacing Unit

Base unit: **8px**

Spacing scale:
- 4px
- 8px
- 16px
- 24px
- 32px
- 48px
- 64px
- 96px

All spacing values must follow this scale.

---

## 2. Header Spacing

### Desktop
- Horizontal padding: 32px–48px
- Vertical padding: 16px–24px
- Distance from hero/banner below: 32px

### Mobile
- Horizontal padding: 16px
- Vertical padding: 12px–16px

Header must feel compact but breathable.

---

## 3. Footer Spacing

### Desktop
- Horizontal padding: 48px
- Vertical padding: 48px–64px
- Column gap inside footer: 32px

### Mobile
- Horizontal padding: 16px
- Vertical padding: 32px
- Section spacing: 24px

Footer should feel structured and balanced.

---

## 4. Product Card Grid — Edge to Edge

### Core Rule

Product grid is **edge-to-edge horizontally**.  
There are **no horizontal outer margins**.  
Cards align directly to container edges.

### Columns

- Mobile: 2 columns
- Small desktop: 4 columns
- Large desktop: 5 columns
- Extra-wide screens: 6 columns

### Horizontal Gaps
- No outer padding (edge-to-edge)
- Internal grid gap: 0px or 1px (if border system requires separation)

### Vertical Gaps
- Row spacing: 24px–32px
- Section spacing above grid: 32px–48px

Vertical rhythm must remain consistent.

---

## 5. Section Spacing

Between major sections:
- 64px–96px (desktop)
- 48px (mobile)

Between title and content:
- 24px–32px

---

## 6. Content Blocks

Text blocks:
- Paragraph spacing: 16px
- Heading to paragraph: 16px–24px
- Heading to section: 32px–48px

---

# OPTION 2 — Standard Store Spacing System

This version uses traditional horizontal gutters and balanced grid spacing.

---

## 1. Page Container

### Desktop
- Max width: 1280px–1440px
- Horizontal padding: 32px–48px

### Mobile
- Horizontal padding: 16px

---

## 2. Product Grid

### Columns
- Mobile: 2 columns
- Tablet: 3 columns
- Desktop: 4 columns
- Large desktop: 5 columns

### Gaps
- Horizontal gap: 16px–24px
- Vertical gap: 24px–32px

Grid must feel breathable but dense enough for fashion retail.

---

## 3. Section Spacing

- Between large sections: 64px (desktop)
- Between medium sections: 48px
- Between small blocks: 24px

Mobile reduces spacing by ~25%.

---

## 4. Cards & Components

### Product Cards
- Inner padding (if used): 16px
- Space between image and title: 12px
- Space between title and price: 8px
- Space between price and actions: 12px

### Buttons
- Inner horizontal padding: 16px–24px
- Inner vertical padding: 12px–16px
- Space between stacked buttons: 12px

---

## 5. Filters & Sorting

- Space between filter groups: 24px
- Space between filter items: 8px–12px
- Distance from filter panel to grid: 32px

---

# Vertical Rhythm Rule

All vertical spacing must follow the 8px system.

Avoid:
- Random spacing values
- Inconsistent section gaps
- Mixed rhythm between pages

---

# Design Principles

Spacing must create:

- Clean visual hierarchy
- Editorial balance
- Product focus
- Premium fashion aesthetic
- Clear scan paths
- Consistent alignment

The layout should feel intentional, structured, and scalable across all devices.
