# Header Alignment & Global Spacing System

---
# 1. Header Alignment Structure

The header must follow a strict alignment structure to maintain visual balance and consistent layout behavior.

### Horizontal Alignment Zones

The header is divided into three primary alignment zones:

**Left Zone**
- Store Location
- Language
- Logotype (Logo)

Elements in this zone must align to the **left side of the header**.

**Center Zone**
- Gender categories: MEN / WOMEN
- Category menu:
  - Shoes
  - Clothing
  - Bags
  - Accessories
  - New Arrivals
  - Sale

These elements must be **horizontally centered relative to the full header width**.

**Right Zone**
- Store Location indicator (if displayed in utility form)
- Search  
- Account  
- Favorites icon 
- Shopping Cart (Bag) icon

Elements in this zone must align to the **right side of the header**.

# 2. Global Side Padding Consistency Rule

The left and right padding of the Header must strictly match the global horizontal spacing used across the entire website.

### Core Rule

- The left edge of Header content must align with all other page content.
- The right edge of Header content must align with all other page content.
- No additional or reduced padding is allowed specifically for the Header.
- The Header must align with the same grid system

### Gender Switch (WOMEN / MEN)

- Must be positioned in the visual center of the header
- Acts as the central anchor of the header layout
- Must remain horizontally centered relative to the full header width

### Utility Icons (Account / Wishlist / Bag)

Utility actions must be displayed as icons only.

Applies to:
- Search  
- Account  
- Feavorites  
- Shopping cart
- 
Rules:

- Only the icon should be visible
- Text labels must not appear in the header
- Icons must be evenly spaced
- Notification badges may appear but must not affect alignment

---

# 4. Equal Spacing Between Blocks

Spacing between major header blocks must be consistent.

Recommended gap scale:
- 20px — tight groups (icons)
- 24px — related elements
- 32px — major zones

---

# 5. Top Bar (Region / Language / Phone)

- Fixed height (40px–48px)
- Horizontal padding must match the main Header padding
- Must align with the same grid system

---

# 6. Category Navigation Bar

Category navigation must follow the same layout grid as the header.

Rules:

- Must align exactly with header padding
- Vertical padding: 12px – 16px
- Navigation items must be vertically centered
- Spacing between categories: 24px – 32px
- The category group must be center-aligned horizontally

Applies to:

- Shoes  
- Clothing  
- Bags  
- Accessories  
- New Arrivals
- Sale  

If SALE is highlighted, it must not break spacing rhythm.

---

# 7. Optical Balance Rules

- Icons must align visually with text baseline
- Badge counters must not affect layout alignment
- Search input height must visually match adjacent blocks
- No element should visually “float” above or below others
