## Conventions

- **Block** — a reusable content section (Hero, NewArrivals, etc.)
- **Field** — an input/textarea with label and placeholder
- **CTA** — a button with an action

## SEO / social networks

Each page file has a **"SEO / social networks"** section with page fields:
- Meta title / description / keywords
- OpenGraph (title / description / image / type / url)
- Twitter Card (card / title / description / image)

Global defaults are described in [global-shell.md](global-shell.md).
